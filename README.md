# Begin Exchange

A modern currency exchange website with live exchange rates.

## Features

- **Live Exchange Rates** - Real-time currency rates from reliable APIs
- **Currency Converter** - Convert between multiple currencies (USD, EUR, SYP, EGP, CNY)
- **Quick Convert** - Fast conversion widget in the sidebar
- **Popular Pairs** - Quick access to common currency pairs
- **Responsive Design** - Works on desktop and mobile devices

## Getting Started

1. Clone the repository
2. Install dependencies:
   
```
bash
   npm install
   
```
3. Run the development server:
   
```
bash
   npm run dev
   
```
4. Open your browser and navigate to the local URL (usually http://localhost:5173)

## Tech Stack

- HTML5
- CSS3
- TypeScript
- Vite
